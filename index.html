    First Post or why starting a technical blog is hard

    Why start a technical blog? As a devops engineer I work with various tools and technologies everyday. Some of them I use everyday or almost everyday (AWS, Terraform, Git , Docker, etc. ) and some only rarely. If you need to use use something you only used a month or two ago chances are you will have to spend some time getting familiar with the tool again. This is where notes come very useful.
